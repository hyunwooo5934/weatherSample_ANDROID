# Introduce

Open Weather API 기반의 Android 날씨 학습 프로젝트
- Open API 기반 날씨 데이터 조회

## Project Goal
- MVVM 아키텍처 적용
- repository 패턴을 통한 데이터 계층 분리
- Room Database를 이용한 로컬 데이터 저장
- Hilt(DI)를 활용한 의존성 주입 구조 이해
- Retrofit + Coroutine 기반 비동기 네트워크 처리

## Tech Stack
Kotlin, MVVM, Repository Pattern, DI(Hilt),Retrofit2, Room
